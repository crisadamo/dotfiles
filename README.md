# Dotfiles


## Install

This includes my configuration for homebrew, ZSH, git, editor, etc

```
make all
```

## Installing with homebrew

```
brew bundle --file=~/.dotfiles/Brewfile
```

and

```
brew bundle --force cleanup --file=~/.dotfiles/Brewfile
```
